## 🎮 ゲーム概要

### コンセプト
- **相撲ルール**を採用したバトルゲーム（押し合いで落とす/倒す）
- 小さなステージで1vs1のシンプルなルール
- 工夫次第で格上にも勝てる戦略性
- **世界観・キャラは自由**（力士に限定されない）

### ゲーム名の由来
- **Sumo**: 世界的に認知されたルール（押し合いで落とす）を表す
- **Neo**: 未来的・新しい印象、ギミックステージとの相性

### ゲームモード

| モード | 説明 | 備考 |
|--------|------|------|
| CPU対戦（ストーリー） | ステージ制で進行 | メインモード |
| PvP（ステータス有り） | 育成成果を発揮 | ランクマッチ |
| PvP（ステータス無し） | 純粋な腕勝負 | 無差別級 |
| 無限モード | 50ステージクリア後の周回 | エンドコンテンツ |

## 🕹️ 操作方法（PC版）

### 基本操作（キーボード完結）

**左手操作（WASD周辺）**

| 操作 | キー | 詳細 |
|------|------|------|
| 移動 | WASD | 前後左右に移動 |
| ジャンプ | Space | ジャンプ |
| 攻撃 | F | 前方に押し攻撃 |
| ガード | Shift | 押している間ガード（スタミナ消費） |
| 左回避 | Q | 左にステップ（スタミナ消費、無敵時間あり） |
| 右回避 | E | 右にステップ（スタミナ消費、無敵時間あり） |
| スキル | R | SP50%以上で発動可能 |
| 操作ガイド | H | 表示/非表示切替 |

**右手操作（十字キー）**

| 操作 | キー | 詳細 |
|------|------|------|
| カメラ左右 | ← → | カメラ回転 |
| カメラ上下 | ↑ ↓ | カメラ角度調整 |

### 指配置イメージ
```
  [Q]左回避  [W]前  [E]右回避  [R]スキル
           [A]左  [S]後  [D]右  [F]攻撃
  [Shift]ガード      [Space]ジャンプ
```
| スキル | F | SP消費して発動（クールダウンあり） |
| カメラ | マウス | 視点を動かす |

### 勝利条件

| 条件 | 説明 |
|------|------|
| リングアウト | 敵を落下させる（Y座標 < -20） |
| KO | 敵のHPを0にする |
| タイムアップ | 時間切れ時にHP残量が多い |

### 敗北条件

| 条件 | 説明 |
|------|------|
| 落下 | 自分が落下する（Y座標 < -20） |
| KO | 自分のHPが0になる |
| タイムアップ | 時間切れ時にHP残量が少ない |

## 📊 パラメーター設計

### パラメーター一覧（8種）

| パラメータ | 説明 | 備考 |
|------------|------|------|
| **HP** | 体力。尽きると負け | 基本生存力 |
| **Power** | 押す力 | 空振りで隙が生まれる |
| **Stability** | 体幹。押されにくさ | Weightではなく体幹で表現 |
| **Speed** | 移動速度 | 走り・歩き両方に影響 |
| **Jump** | ジャンプ力 | 高所への移動、回避に使用 |
| **Guard** | ガード時のダメージ軽減率 | 防御の強さ |
| **Stamina** | スタミナ最大値＋回復速度 | 連打ゲー防止 |
| **SP** | スキルゲージ | 最大値と回復速度 |

### スタミナシステム

| 状況 | スタミナ消費 |
|------|--------------|
| 押す（ヒット） | 小 |
| 押す（空振り） | 大（ペナルティ） |
| ガード | 中（継続消費） |
| 回避 | 中 |
| スタミナ0 | **息切れ状態**（数秒動けない） |

### 設計意図
- **Power vs Stability**: 押す力と押されにくさを分離
- **Stamina制**: 連打ゲーを防止、戦略性を追加
- **空振りペナルティ**: 考えなしの攻撃を抑制

### レベルシステム

| 項目 | 内容 |
|------|------|
| 経験値獲得 | バトル勝利/敗北で獲得 |
| 必要経験値 | 100 × 1.2^(Lv-1) |
| レベルアップ効果 | HP+5, Power+2, Stability+2 |

### 強化ショップ

コインを使ってパラメーターを強化できる。

| 強化項目 | 効果 | 初期コスト | コスト増加 | 最大Lv |
|----------|------|------------|------------|--------|
| ❤️ HP | +10 | 50 | +25 | 10 |
| ⚔️ 攻撃力 | +2 | 100 | +50 | 10 |
| 🛡️ 安定性 | +2 | 100 | +50 | 10 |
| 💨 速度 | +1 | 150 | +75 | 5 |

※レベルアップ効果と強化効果は**加算**される

## ⚡ スキルシステム

### 基本仕様

- 1プレイヤー**1スキル**装備可能
- 対戦中は入れ替え不可
- 対戦前にスキルを選択

### スキルゲージ（SP）

| 項目 | 値 | 説明 |
|------|-----|------|
| 最大値 | 100 | SPの最大値 |
| 初期値 | **0** | バトル開始時は0からスタート |
| 回復速度 | 5/秒 | 自動で徐々に回復 |
| 発動条件 | **50%以上** | SPバー上の黄色ラインで表示 |

### スキル発動
- SPが50%（50）以上で発動可能
- SPバー上に発動ラインを表示
- 発動時にSPを消費

### スキル詳細一覧（10種類）

| # | 名前 | 効果 | SP消費 | クールダウン |
|---|------|------|--------|--------------|
| 1 | **Power Push** | 押す力2倍 | 50% | 5秒 |
| 2 | **Double Jump** | 空中ジャンプ可能（10秒間） | 50% | 5秒 |
| 3 | **Absolute Defense** | 完全無敵（動けない、3秒） | 50% | 15秒 |
| 4 | **Speed Boost** | 移動速度2倍（8秒） | 50% | 10秒 |
| 5 | **Invisible** | 透明化（8秒、攻撃/被ダメで解除） | 50% | 15秒 |
| 6 | **Attract** | 相手を自分に引き寄せる（近距離） | 50% | 10秒 |
| 7 | **Shockwave** | 周囲を大きく吹き飛ばす（広範囲・強威力） | 70% | 15秒 |
| 8 | **Heal** | HP35%回復（即時） | 50% | 20秒 |
| 9 | **Barrier** | ダメージ50%軽減＋ノックバック無効（8秒） | 70% | 20秒 |
| 10 | **Teleport** | ステージ中心に瞬間移動（移動中無敵） | 70% | 15秒 |

### スキルのレア度（性能差あり）

| レア度 | 性能倍率 | 見た目 | ガチャ確率 |
|--------|----------|--------|------------|
| コモン | 100% | シンプル | 60% |
| レア | 150% | エフェクト付き | 30% |
| エピック | 200% | 派手 | 8% |
| レジェンド | 300% | 超派手 | 2% |

### スキルのタイプ（同スキルでも4種類）

| タイプ | 特徴 |
|--------|------|
| Aタイプ（威力） | 一撃が重い |
| Bタイプ（効果時間） | 効果が長い |
| Cタイプ（回転） | クールダウンが短い |
| Dタイプ（万能） | 全部少しずつUP |

### スキルの総数

**10スキル × 4レア度 × 4タイプ = 160通り**

### スキル入手方法

| 方法 | 対象 |
|------|------|
| 初期解放 | 基本スキル（コモン3種） |
| ステージクリア報酬 | コモン・レア |
| ゲーム内通貨で購入 | コモン・レア |
| **ガチャ** | **全レア度（レジェンド含む）** |

## 📝 補足・注意事項

### 開発上の注意
- 物理演算ベースの戦闘は**バランス調整が難しい**
- MVP段階で十分にテストプレイすること
- PvPのマッチメイキングは設計が複雑なので注意
- スキルの性能差（3倍）は**プレイヤースキルで逆転可能**な範囲に調整

### 成功のポイント
1. **コアループがシンプルで面白い**こと
2. **課金動機を明確に**すること（レジェンドAタイプが欲しい！）
3. **段階的にリリース**すること
